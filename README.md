# 🏠 Estif Home 1000X - Enterprise Smart Home System

[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)
[![Node.js](https://img.shields.io/badge/Node.js-18.x-green.svg)](https://nodejs.org/)
[![ESP32](https://img.shields.io/badge/ESP32-Compatible-red.svg)](https://www.espressif.com/)

A complete, production-ready smart home automation system with voice control, AI-powered automation, and global remote access. Control your home from anywhere in the world using your voice or web interface.

## ✨ Features

### 🎮 Device Control
- **Real-time Control** - Instant device toggling with WebSocket updates
- **6 Device Support** - Light, Fan, AC, TV, Heater, Water Pump
- **Custom GPIO** - Light:23, Fan:22, AC:21, TV:19, Heater:18, Pump:5

### 🎤 Voice Commands
- **Bilingual Support** - English & Amharic (አማርኛ)
- **Wake Words** - "Hey Estiph" (English) / "ሰላም እስቲፍ" (Amharic)
- **Natural Language** - "Turn on living room light" / "መብራት አብራ"

### 🤖 Smart Automation
- **Auto Mode** - Temperature-based AC/Heater control
- **Schedules** - Time-based device automation
- **Smart Scenes** - One-tap scenes (Good Morning, Movie, Party, etc.)
- **AI Suggestions** - Energy-saving recommendations

### 📊 Monitoring & Analytics
- **Energy Tracking** - Real-time power consumption
- **Usage Statistics** - Daily, weekly, monthly reports
- **Device Health** - Predictive maintenance alerts

### 🌍 Remote Access
- **Global Control** - Access from anywhere with internet
- **WebSocket** - Real-time updates
- **Secure** - JWT authentication, rate limiting

### 📱 Mobile Optimized
- **Responsive Design** - Works perfectly on phones
- **PWA Support** - Install as app on your phone
- **Touch Friendly** - Large buttons for easy tapping
